## Overview

CP-2 is a comprehensive developer toolkit designed to facilitate AI-driven educational content creation and management. It combines a lightweight frontend with powerful backend APIs to automate content generation, document summarization, and document processing. Built with a modular architecture, CP-2 enables seamless integration into existing systems and accelerates the development of interactive learning platforms.

### Why CP-2?

This project empowers developers to build intelligent educational applications with ease. The core features include:

- **Question Generation:** Leverages transformer models like T5 to produce relevant, context-aware questions from textual content.
- **Content Summarization:** Uses advanced NLP techniques to generate concise summaries of lengthy documents.
- **Robust API Endpoints:** Facilitates document uploads, question creation, and content analysis with scalable backend services.
- **Lightweight Frontend:** Built with vanilla HTML, CSS, and JavaScript for simple, effective user interface.
- **Extensible Architecture:** Modular design supports future enhancements and integrations for diverse educational needs.

## Getting Started

### Prerequisites

This project requires the following dependencies:

- **Programming Language:** JavaScript
- **Package Manager:** Pip, Npm

### Installation

Build CP-2 from the source and install dependencies:

1. Clone the repository:

```bash
git clone https://github.com/MEETSHARM36/CP-2
```

2. Navigate to the project directory:

```bash
cd CP-2
```

3. Install the dependencies:

Using pip:
```bash
pip install -r backend/requirements.txt
```

Using npm
```bash
npm install
```
